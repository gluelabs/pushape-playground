# Pushape Playground 


## Installation

Run `npm install` inside the project.

## Environment

Remember to add these two variables in `enviroment.ts` and `environment.prod.ts`:

```
  sender_id: '<YOUR_SENDER_ID>',
  pushape_app: '<YOUR_PUSHAPE_APP_ID>'
```
